This class returns info about the Assemblies as a tree of AssemblyNames. It is not thread safe since we keep an internal list of assemblynames to avoid stack overflow due to circular reference.

ReferencedAssemblyNames
This is the list of referenced AssemblyNames. If the AssemblyName has none referenced AssemblyName then this list is empty. Note: If the AssemblyName is subject to circular reference, i.e. we have it somewhere else in the AssemblynameNode tree then this property is null. If we didn't stop the recursion we would be circular reference. (yes - the framework contains at least one circular reference)
